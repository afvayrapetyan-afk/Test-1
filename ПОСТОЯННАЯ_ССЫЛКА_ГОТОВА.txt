1. Нажмите "Continue with GitHub"

2. Подтвердите доступ к GitHub репозиториям

3. Найдите в списке: "Test-1" или "afvayrapetyan-afk/Test-1"

5. На странице конфигурации:
   ⚠️  ВАЖНО! Измените настройки:

   Build and Output Settings:
   - Framework Preset: Vite
   - Build Command: cd frontend && npm install && npm run build
   - Output Directory: frontend/dist
   - Root Directory: ./

   (Если Vercel определит автоматически - просто нажмите Deploy)

6. Нажмите "Deploy"

7. Ждите 2-3 минуты
